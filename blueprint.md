# Word Photo Reader App Blueprint

## 1. 개요 (Overview)

이 프로젝트는 사용자가 단어가 포함된 이미지를 업로드하거나 카메라로 직접 촬영하면, 이미지 속에서 한 단어를 자동으로 인식하여 원어민 발음으로 5번 반복해서 읽어주는 웹 애플리케이션입니다. 모바일 환경에 최적화된 반응형 디자인을 갖추고 있으며, 시각적으로 매력적이고 사용하기 쉬운 인터페이스를 제공하는 것을 목표로 합니다.

## 2. 디자인 및 스타일 (Design & Style)

- **레이아웃:** 화면 중앙에 핵심 기능을 담은 카드 형태의 UI를 배치하여 사용자가 기능에 집중할 수 있도록 합니다.
- **색상 팔레트:** 생동감 있고 활기찬 색상 조합을 사용하여 긍정적인 사용자 경험을 제공합니다.
- **타이포그래피:** 가독성 높은 폰트를 사용하고, 정보의 중요도에 따라 폰트 크기를 조절하여 명확한 정보 전달을 돕습니다.
- **시각 효과:** 카드와 버튼에 부드러운 그림자 효과를 적용하여 입체감을 주고, 사용자의 인터랙션에 시각적 피드백을 제공합니다.
- **반응형 디자인:** 모바일, 태블릿, 데스크탑 등 다양한 화면 크기에서 최적의 사용성을 제공합니다.

## 3. 기능 (Features)

- **두 가지 이미지 입력 방식:**
    - **파일 업로드:** 사용자는 자신의 기기에서 단어가 포함된 이미지 파일(JPG, PNG 등)을 선택하여 업로드할 수 있습니다.
    - **카메라 촬영:** 사용자는 기기의 카메라를 직접 사용하여 단어가 포함된 사진을 찍어 업로드할 수 있습니다.
- **광학 문자 인식 (OCR):**
    - Tesseract.js 라이브러리를 사용하여 브라우저 내에서 이미지의 텍스트를 인식합니다.
    - 서버 없이 클라이언트 측에서 모든 처리가 이루어져 빠르고 안전합니다.
    - 이미지 내에서 다른 단어와 분리된 단일 단어를 우선적으로 추출합니다.
- **텍스트 음성 변환 (TTS):**
    - 웹 표준 API인 \`SpeechSynthesis\`를 사용하여 추출된 단어를 음성으로 변환합니다.
    - 'en-US' (미국 영어)를 기본 언어로 설정하여 원어민에 가까운 발음을 제공합니다.
- **반복 재생:**
    - 인식된 단어를 2초 간격으로 총 5번 반복하여 읽어줍니다.
    - 사용자가 단어의 발음을 충분히 듣고 학습할 수 있도록 돕습니다.
- **상세한 진행 상태 표시:** OCR 엔진 초기화부터 언어 파일 로딩, 텍스트 인식률(%)까지 사용자에게 구체적인 처리 과정을 보여줍니다.

## 4. 현재 진행 계획 (Current Plan)

**완료됨: 사진 업로드 및 카메라 촬영 기능 추가**

1.  **HTML (`index.html`):**
    - "Upload Image"와 "Use Camera" 버튼을 추가하고, 아이콘으로 시각적 효과를 더했습니다.
    - 이미지 미리보기 영역과 카메라 영상(`<video>`) 및 캡처를 위한 `<canvas>` 영역을 추가했습니다.
2.  **CSS (`style.css`):**
    - 새로운 버튼들과 이미지 미리보기 영역에 기존 디자인과 통일성 있는 스타일을 적용했습니다.
    - 카메라 사용 시 버튼 UI가 자연스럽게 전환되도록 스타일을 추가했습니다.
3.  **JavaScript (`main.js`):**
    - "Upload Image" 버튼 클릭 시 파일 선택 창이 열리는 기능을 구현했습니다.
    - "Use Camera" 버튼 클릭 시 카메라 권한을 요청하고, 영상 스트림을 `<video>`에 표시하는 기능을 구현했습니다.
    - "Take Photo" 버튼으로 비디오 화면을 캡처하여 이미지 파일로 변환하는 로직을 작성했습니다.
    - 파일 업로드와 카메라 캡처 두 가지 방식 모두에서 생성된 이미지를 처리할 수 있도록 로직을 통합했습니다.
